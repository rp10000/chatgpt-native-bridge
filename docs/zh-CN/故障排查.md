# 故障排查

## `npx` 找不到

先安装 Node.js 和 npm。项目要求 Node 18 或更新版本。

## npm 包还没发布

发布到 npm registry 前，请用：

```bash
npx --yes --package github:rp10000/chatgpt-native-bridge cgn setup --mcp
```

或本地安装：

```bash
git clone https://github.com/rp10000/chatgpt-native-bridge.git
cd chatgpt-native-bridge
npm link
cgn --help
```

## Codex 里没有 MCP server

先安装 Codex MCP 配置：

```bash
npx --yes --package github:rp10000/chatgpt-native-bridge cgn setup --mcp
```

然后重启 Codex，或打开一个新的 Codex 线程。

如果你使用手动 HTTP server 备用路径，再启动本地 server：

```bash
cgn mcp serve --host 127.0.0.1 --port 47832
```

然后打开：

```text
http://127.0.0.1:47832/health
```

应该看到一个简短 JSON health 响应。

## MCP 端口被占用

换一个端口：

```bash
cgn mcp serve --host 127.0.0.1 --port 47833
cgn mcp config --port 47833
```

ChatGPT 端要使用对应的 `/mcp` endpoint。

## ChatGPT 访问不了 `127.0.0.1`

有些 ChatGPT MCP 连接方式不能直接访问你的本机 loopback 地址。请使用账号或工作区支持的官方 MCP 连接方式，例如 Secure MCP Tunnel。

不要用抓取 ChatGPT 网页、提取 cookies、读取 localStorage 或隐藏接口来解决。

## 我以为 MCP 会有 shell 工具

不会。这个项目让 ChatGPT 通过受限 MCP 工具提供建议，再由 Codex 在本地执行。

查看当前 MCP 工具：

```bash
cgn mcp config
```

## `cgn open` 没打开浏览器

先 dry-run：

```bash
cgn open latest --dry-run
```

然后手动打开 `https://chatgpt.com`，复制 outbox 里的 `01_PASTE_TO_CHATGPT.md`。

如果想只看路径：

```bash
cgn open latest --mode manual
```

如果想同时打开 ChatGPT 和 outbox 文件夹：

```bash
cgn open latest --mode auto
```

## 剪贴板复制失败

打开 `cgn open` 输出的 outbox 路径，手动复制 `01_PASTE_TO_CHATGPT.md`。

## 看不到应该上传什么

运行：

```bash
cgn open latest --mode manual
```

看输出里的：

```text
Paste prompt
Upload/select in ChatGPT
Outbox
```

## 找不到 latest handoff

先看状态：

```bash
cgn status
```

如果没有 handoff，先创建：

```bash
cgn handoff --task "Review onboarding UX" --type plan,ux-review
```

## ChatGPT 回复没有导入

复制 ChatGPT 回复后运行：

```bash
cgn done
```

或保存成文件：

```bash
cgn import latest ./reply.md
```

## Codex 没有使用 Skill

明确告诉 Codex：

```text
Use chatgpt-native-bridge for this task.
```

并检查：

```bash
npx --yes --package github:rp10000/chatgpt-native-bridge cgn doctor
```

## secret guard 拦截了文件

说明文件路径或内容看起来像敏感信息。请删除敏感内容、写安全摘要，或不要把该文件放进 handoff。
