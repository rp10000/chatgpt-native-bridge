# 使用场景

## 适合使用

- 大改前规划
- 架构复核
- diff 二次审查
- UI/UX 截图复核
- 命名、文案、产品判断
- 图片方向和提示词
- 研究和资料整理
- README、发布说明和定位优化

## 不适合使用

- 需要立即本地改代码的小修小补
- 需要保密且不能交给 ChatGPT 的内容
- 需要自动提交、自动 push、自动控制网页的流程

## 推荐路径

Pro 深度规划：

```text
客户端 -> Pro -> 复制回复 -> 客户端写回 Codex
```

Thinking 工具复核：

```text
客户端启动 MCP -> ChatGPT Thinking 调工具 -> 写回 Codex
```

手动备用：

```text
cgn handoff -> ChatGPT -> cgn done
```
