# 使用场景

## 适合使用

- 大改前规划
- 架构复核
- diff 二次审查
- UI/UX 截图复核
- 命名、文案、产品判断
- 图片方向和提示词
- 研究和资料整理
- README、发布说明和定位优化

## 不适合使用

- 只需要立刻本地改代码的小修小补
- 需要保密且不能交给 ChatGPT 的内容
- 需要自动提交、自动 push、自动控制网页的流程

## 推荐路径

主路径：

```text
客户端连接 ChatGPT -> ChatGPT 网页端读写项目 -> 生成交接报告
```

辅助路径：

```text
客户端打包上下文 -> Pro 辅助规划 -> 客户端导入备用说明
```

手动备用：

```text
cgn handoff -> ChatGPT -> cgn done
```
